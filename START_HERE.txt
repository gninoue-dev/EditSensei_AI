🚀 DÉMARRAGE RAPIDE (5 MINUTES)

ÉTAPE 1: Extraire le ZIP
    unzip ÉDITSENSEI_AI_v1.0.0.zip
    cd ÉDITSENSEI_AI

ÉTAPE 2: Backend
    cd backend
    python3 -m venv venv && source venv/bin/activate
    pip install -r requirements.txt
    cp .env.example .env
    
    ⚠️  IMPORTANT: Éditer .env et ajouter OPENAI_API_KEY
    
ÉTAPE 3: Frontend
    cd ../frontend
    npm install

ÉTAPE 4: Démarrer
    cd ..
    ./start.sh  (Unix/Mac)
    # ou
    start.bat   (Windows)

ÉTAPE 5: Accéder
    🌐 Frontend: http://localhost:5173
    🔧 Backend:  http://localhost:8000
    📚 API Docs: http://localhost:8000/docs

🆘 PROBLÈMES COURANTS

Q: "ImportError: No module named 'openai'"
A: Exécuter: pip install -r requirements.txt

Q: "Port 8000 already in use"
A: Changer le port dans backend/.env ou tuer le processus

Q: "Module not found"
A: Vérifier que npm install a été lancé dans frontend/

Q: "OPENAI_API_KEY not set"
A: Éditer backend/.env et ajouter votre clé
